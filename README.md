# **Student Result Analysis**

## **Overview**
This project analyzes a dataset of **30,641 students** to explore factors influencing academic performance.  
Tech stack used: **Python, Pandas, NumPy, Matplotlib, Seaborn**.

## **Conclusion**
- **Parental education** strongly influences student performance.  
- **Marital status** has negligible impact.  
- **Gender ratio** is balanced with slight female majority.  
- **Ethnic distribution** shows Group C as most represented.
